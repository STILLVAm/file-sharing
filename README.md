# 个人文件云存储系统

安全、高效的个人文件存储解决方案，支持大文件上传和用户认证。

## 功能特点

- ✅ 10GB大文件上传
- 🔒 用户认证与API密钥保护
- 📁 文件管理（上传、下载、删除）
- 📊 仪表盘与文件列表
- ⚡ 断点续传与分片上传
- 🔄 自动备份与密钥轮换
- 🚀 一键部署阿里云服务器

## 技术栈

### 后端
- Python Flask
- Gunicorn
- JWT认证
- 分片文件处理

### 前端
- Vue 3
- Vue Router
- Vuex
- Element Plus
- Axios

### 基础设施
- Docker
- Nginx
- PostgreSQL（可选）
